Create a virtual environment and install dependencies:
Should have python version < 3.11 
   ```bash
   python -m venv .venv
   source .venv/bin/activate  # Windows: .venv\Scripts\activate
   pip install -r requirements.txt
   ```
Run the app:
   ```bash
   python run.py
   ```
   Then open http://127.0.0.1:5000/
